# Framing the Machine

*""Framing the Machine: A 25-Year Temporal and Sentiment Analysis of Artificial Intelligence Representation in Mainstream Journalism Across the US (2000-2025)""*. 

## Prerequisites
1. **Python** installed on your system.
2. **Kaggle Dataset**: Download the `nyt-metadata.csv` file from [Kaggle: NYT Articles 2.1M+ (2000-Present)](https://www.kaggle.com/datasets/aryansingh0909/nyt-articles-21m-2000-present). Place the extracted `nyt-metadata.csv` file directly into the root directory of this project.
